Quick Start

1. Extract Digger/ into YourProject/Plugins/ 2. Open project in UE5 — click Yes to compile 3. Open Digger Editor Mode from the Modes toolbar 4. Click Setup Landscape for Dynamic Holes 5. Select dig tool and click the landscape to dig!
